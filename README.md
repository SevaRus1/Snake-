Snake Game

Классическая игра Змейка на Python с библиотекой Arcade.

Управление:
- Пробел - старт
- Стрелки - движение
- R - рестарт
- ESC - выход в меню

Типы еды:
- Красная (+1 очко, 70%)
- Жёлтая (+3 очка, 20%)
- Фиолетовая (+5 очков, 10%)
Авторы: Мягков, Еремин
Чтобы запустить разорхивируйте на рабочем столе и нажмите два раза по main.py
